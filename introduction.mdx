## What is this project?

`python-mariadb-example` is a command-line application that shows how to connect a Python program to a MariaDB database. It uses the official `mariadb` Python connector and organizes the code using the **Model-View-Controller (MVC)** pattern.

The app operates on the classic **scott/tiger** demo schema вЂ” the same two-table dataset (employees and departments) used in Oracle training materials for decades. It lets you list, look up, insert, and delete records through an interactive CLI menu.

## What you'll learn

- How to establish and manage a MariaDB connection in Python using `MariaDBInstance`
- How to structure database access code with MVC separation of concerns
- How to use Python dataclasses as lightweight model objects
- How to write parameterized SQL queries to prevent injection
- How to build an interactive CLI loop with `match` statements (Python 3.10+)

## The scott/tiger demo database

The app connects to a database named `scott` with the user `scott` and password `tiger`. This is a well-known demo schema originally shipped with Oracle databases. It contains two tables:

- **`emp`** вЂ” employee records with fields like name, job title, salary, manager, and department
- **`dept`** вЂ” department records with a name and location

The schema is small enough to understand at a glance but rich enough to demonstrate joins, aggregations, and relational lookups.
