## Key features

- **Parameterized queries** — All SQL uses `?` placeholders to prevent injection vulnerabilities
- **MVC separation** — Models, controllers, and UI each have distinct responsibilities
- **Typed results** — Query results are mapped to Python dataclasses for safe attribute access
- **Cancellable inputs** — Any multi-field operation can be aborted mid-way with `/cancel`
- **JOIN queries** — Department controller demonstrates a `LEFT JOIN` to count employees per department

<Steps>
  <Step title="Set up the database">
    Install MariaDB and create the `scott` database with the `emp` and `dept` tables. See [Prerequisites](/prerequisites) for details.
  </Step>
  <Step title="Install dependencies">
    Run `pip install -r requirements.txt` to install the `mariadb` Python connector.
  </Step>
  <Step title="Configure the connection">
    Update the credentials in `main.py` to match your MariaDB user and database.
  </Step>
  <Step title="Run the app">
    Execute `python main.py` and interact with the CLI menu to query and manage data.
  </Step>
</Steps>
